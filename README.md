# LineStock

A real-time stock price checking linebot for Taiwan market

# Environment
- Python
- Heroku
- line-bot-sdk for python

# Usage
1. Type the stock code which you want to check in chatroom.
2. The line bot will reply the stock information:
'''
股票代號
昨收價
漲跌幅
當前價
'''
